:::step{number=1 icon=LifeBuoy title="View Your Tickets" navigation="Sidebar > Support"}
The support page shows all your help requests. You can see which ones are still open (being worked on) and which ones have been resolved. Think of it as your help desk inbox.
:::

:::screenshot{caption="Your Support tickets page"}
:::

:::step{number=2 icon=Plus title="Create a New Ticket" navigation="Support > Create Ticket"}
Click "Create Ticket" and describe your problem or question. Be as specific as you can:

- What were you trying to do?
- What happened instead?
- Any error messages you saw?
- When did it start happening?
  :::

:::screenshot{caption="Your Create Ticket form"}
:::

:::step{number=3 icon=MessageSquare title="Follow Up" navigation="Click on any ticket"}
Click on a ticket to see the conversation. You can add more information, answer questions from the support team, or confirm that your issue has been resolved.
:::

:::step{number=4 icon=Lightbulb title="Tips for Faster Help" navigation="Keep these in mind"}

- Include screenshots of error messages
- Mention which server or resource is affected
- Describe the steps that led to the problem
- Check if the issue is already mentioned in these docs
  :::

:::callout{type=info}
Most questions about how to use the platform are answered right here in this documentation. Browse through the topics in the sidebar -- you might find your answer without needing to wait for a support response!
:::
