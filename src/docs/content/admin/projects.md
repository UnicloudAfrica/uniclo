:::step{number=1 icon=FolderKanban title="View All Projects" navigation="Sidebar > Infrastructure > Projects"}
The projects page lists every project across all tenants. Each project belongs to a specific tenant and contains instances, networks, storage, and other resources. You can search by project name or filter by tenant to find what you need.
:::

:::screenshot{caption="Your Projects list showing all tenants"}
:::

:::step{number=2 icon=Plus title="Create a Project" navigation="Sidebar > Infrastructure > Projects > Create Project"}
Click the create button to set up a new project. Select which tenant it belongs to, give it a name, and choose the region. Projects help keep infrastructure organized — like having separate folders for different departments.
:::

:::step{number=3 icon=Eye title="Project Details" navigation="Click any project in the list"}
Click on a project to see everything inside it — instances, networks, storage volumes, and more. This is your quick overview of all the resources grouped under one project.
:::

:::callout{type=tip}
Encourage tenants to use separate projects for different environments (development, staging, production). This keeps things organized and makes it easier to manage access and billing.
:::
